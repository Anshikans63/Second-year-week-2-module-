# Second-year-week-2-module-

 Searching and Sorting Algorithms in C 

This repository contains the solutions for the Week 2 Module of the Data Structures and Algorithms (DSA) lab. The programs are written in C language and demonstrate the implementation of basic searching and sorting algorithms using arrays. The module includes Linear Search, Binary Search, Bubble Sort, Insertion Sort, and Selection Sort with real-life problem statements such as searching student roll numbers, product IDs, arranging exam scores, library book numbers, and ranking player scores.

Algorithms Covered

- Linear Search
- Binary Search
- Bubble Sort
- Insertion Sort
- Selection Sort

Concepts Used

- Arrays
- Loops ("for", "while")
- Conditional Statements ("if-else")
- Functions of Searching and Sorting
- Swapping of Elements
- Time Complexity Basics

Features

- Simple and beginner-friendly C programs.
- Well-structured logic with real-world examples.
- Covers both searching and sorting techniques.
- Easy to understand and execute.
- Suitable for academic assignments and DSA practice.

Learning Outcomes

After completing this module, you will be able to:

- Search elements using Linear Search and Binary Search.
- Sort data using Bubble Sort, Insertion Sort, and Selection Sort.
- Understand the working and differences between various searching and sorting algorithms.
- Analyze the basic time complexity and efficiency of each algorithm.

Language Used

- C Programming Language

This module strengthens the fundamentals of Data Structures and Algorithms by providing practical implementations of commonly used searching and sorting techniques in C.
